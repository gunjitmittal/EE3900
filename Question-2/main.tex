\section{Question}
\textbf{Question}\\
Determine whether the following signal is periodic. If it is periodic state the period.
\begin{equation}
  x[n] = e^{\frac{jn\pi}{\sqrt2}}
\end{equation}
\section{Solution}
\solution\\
$x[n]$ is periodic with period N if $x[n] = x[n+N] $ for some integer N.\\
\begin{align}
  x[n] &= x[n+N]\\
  \implies e^{\frac{jn\pi}{\sqrt2}} &= e^{\frac{j(n+N)\pi}{\sqrt2}}\\
  &=e^{\frac{jn\pi}{\sqrt2}+\frac{jN\pi}{\sqrt2}}\\
  &=e^{\frac{jn\pi}{\sqrt2}+j 2 \pi k}
\end{align}
For integers N,k
\begin{align}
  2 \pi k = \frac{N\pi}{\sqrt2}\\
  N = 2 \sqrt2 k
\end{align}
There is no k for which N is an integer \\
$\therefore$ The given signal is not periodic
